how-to-html
===========

```html
<tag attribute name="value" style="" height="300" width="200" title="Neela">
  Enclose Body Or Content = {other <tags> or text}
  Some Tags Don't support body or content!
</tag>
```

In a style sheet:
tag{
  name:value;
}

Unlike HTML tags:
```html
<open_tag name="value"></close_tag>
```
